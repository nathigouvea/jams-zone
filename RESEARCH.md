## 3. What Makes Templates Sell

### Pricing Sweet Spots
- **Impulse buy:** $9-$15 (minimal friction, good for bundles)
- **Standard:** $19-$29 (sweet spot for HTML/Framer templates)
- **Premium:** $39-$79 (must justify with exceptional quality + docs)
- **Bundles:** 2-3x single price ($49-$99) — customers feel they're getting a deal

### Preview Quality
- **Live demo link is essential** — products with live preview get 3-5x more conversions
- Animated GIFs in product thumbnails increase CTR by ~40%
- Mobile mockups alongside desktop show responsiveness
- 8+ screenshots minimum; start with the hero/most impressive view
- Dark and light mode previews if applicable

### Description Length & Structure
- Optimal: 400-800 words
- Lead with value proposition, not features
- Bullet list of included files, tech stack, features
- FAQ section reduces support tickets and increases trust
- "What's included" section with file count
- Update history builds trust over time

### Categories & Tags
- Use all available tags/categories
- "HTML Template" + "Landing Page" + "SaaS" = 3x discoverability
- Include your tech stack as tags (Tailwind, Vanilla JS, etc.)
